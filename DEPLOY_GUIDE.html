Deploy to Cloud

Get a permanent public URL for your VN Routing Tool so anyone can open it in their browser — no installation needed. Choose your platform below.

1
Make sure your project folder looks like this
Your data files must be inside a data/ subfolder. The deploy config files go in the root.
your-project/ ├── app.py ├── routing_engine.py ├── preprocess.py ├── poi_loader.py ├── requirements.txt (use the new pinned version) ├── render.yaml (from deploy files) ├── Procfile (from deploy files) ├── runtime.txt (from deploy files) └── data/ ├── test_road.geojson └── VN Sample Data.csv
2
Push your project to GitHub
If you don't have a GitHub repo yet, create one at github.com/new then run these commands inside your project folder:
terminal
git init
git add .
git commit -m "initial commit"
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git push -u origin main
⚠Your data files (GeoJSON + CSV) must be committed to GitHub too — Render can't access files that aren't in the repo.
3
Sign up and create a new Web Service on Render
Go to render.com, sign up with your GitHub account, then:
  1. Click New + → Web Service
  2. Click Connect a repository and pick your repo
  3. Render will auto-detect render.yaml and fill in the settings
  4. Confirm Build Command: pip install -r requirements.txt
  5. Confirm Start Command: uvicorn app:app --host 0.0.0.0 --port $PORT
  6. Set plan to Free
  7. Click Create Web Service
4
Wait for the build to finish (~3–5 minutes)
Render installs your dependencies and starts the server. You'll see live build logs. When it says Your service is live you're done.
✓Render gives you a URL like https://vn-routing-tool.onrender.com. Send /map appended to that URL to your reviewers.
ℹOn the free plan Render spins down after 15 minutes of inactivity. The first visit after sleep takes ~30 seconds to wake up — this is normal.

3
Deploy on Railway
Go to railway.app, sign up with GitHub, then:
  1. Click New Project → Deploy from GitHub repo
  2. Select your repository
  3. Railway auto-detects the Procfile and starts building
  4. Once deployed, click Settings → Domains → Generate Domain
⚠Railway's free tier gives $5 credit/month. Your app will run for roughly 500 hours free — more than enough for sharing and review.
4
Share the /map URL
After Railway assigns your domain, append /map and share it.
✓Unlike Render, Railway does not spin down on inactivity — it stays live as long as you have free credit.
